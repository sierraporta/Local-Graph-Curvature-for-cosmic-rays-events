# Local-Graph-Curvature-for-cosmic-rays-events
We introduce the local Forman-Ricci curvature of visibility graphs as a novel descriptor of cosmic ray dynamics and its coupling with space weather variability. 

## Requirements

- Python ≥ 3.9
- numpy, scipy, networkx, matplotlib, scikit-learn
